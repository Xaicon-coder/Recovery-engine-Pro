🔴 PROBLEMA COMUNE: "Il programma è già in esecuzione"
───────────────────────────────────────────────────────────────────

Se vedi questo errore durante l'installazione:

    "Impossibile installare: programma già in esecuzione"
    "File Recovery Pro.exe è in uso"
    
ANCHE SE l'app non è visibile, ci sono processi residui!


✅ SOLUZIONE RAPIDA (3 PASSI):
───────────────────────────────────────────────────────────────────

PASSO 1: CHIUDI TUTTI I PROCESSI
─────────────────────────────────
a) Apri Task Manager (Ctrl + Shift + Esc)
b) Vai su "Dettagli"
c) Cerca e termina:
   • electron.exe
   • node.exe
   • File Recovery Pro.exe
   
   (Click destro → Termina processo)


PASSO 2: USA LO SCRIPT DI PULIZIA
──────────────────────────────────
Doppio click su uno di questi:

   📁 scripts\cleanup-windows.bat       (FACILE)
   📁 scripts\cleanup-windows.ps1       (POTENTE)

Attendi che finisca e dice "Cleanup completato!"


PASSO 3: RIPROVA INSTALLAZIONE
───────────────────────────────
Ora puoi installare senza problemi!


═══════════════════════════════════════════════════════════════════


🚀 INSTALLAZIONE NORMALE (Se non hai problemi)
───────────────────────────────────────────────────────────────────

METODO 1 - Build da sorgente (RACCOMANDATO):
────────────────────────────────────────────
1. Apri Prompt Comandi in questa cartella
2. Esegui:
   npm install
   npm run dev

3. Per creare l'installer:
   npm run build


METODO 2 - Usa installer pre-built (Se disponibile):
────────────────────────────────────────────────────
1. Cerca file .exe nella cartella release/
2. Doppio click per installare
3. Se errore → vedi sopra "PROBLEMA COMUNE"
